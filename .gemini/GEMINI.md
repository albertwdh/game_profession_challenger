# Profession Challenger (职业挑战者) Project Memory

## 1. 项目愿景 (Project Vision)
一款 2D 职场模拟游戏，通过“资源管理 + 随机事件抉择”的玩法，让玩家体验不同职业的百味人生。核心魅力在于强烈的代入感、黑色幽默以及对现实职场的解构。

## 2. 核心设计理念 (Core Design Philosophy)
**“好玩、有梗、能学到东西，拒绝纯粹的枯燥数值博弈。”**
- **不仅是数值，更是故事:** 每一个事件选项不仅带来数值上的增减，更必须触发一段对应的反馈剧情（Feedback Text），让玩家看到自己选择的真实后果（或啼笑皆非的转折）。
- **硬核科普与职场避坑:** 事件中巧妙植入真实的职业知识（如：什么是 CORS 跨域、如何有效复盘、怎样识别领导的 PUA）。玩家在失败中不仅是被“扣血”，而是学到了一个真实的行业冷知识或生存法则。
- **意料之外的连锁反应:** 引入隐藏状态或物品机制（如“老板画的饼”、“一杯续命咖啡”），让前几天的选择在后续事件中产生意想不到的互动，增加策略深度和重玩价值。

## 3. 核心机制 (Core Mechanics)
- **核心循环 (Core Loop):** 开启新的一天 -> 触发日常/随机事件 -> 做出选择 -> 资源结算 -> 下班活动（加班/娱乐/学习） -> 进入下一天。
- **四大核心资源:**
  - 💰 **存款 (Money):** 归零则破产。
  - ❤️ **精神状态 (SAN/Mental):** 归零则猝死/抑郁。
  - 💼 **业务能力 (Competence):** 影响升职加薪。
  - 🐟 **摸鱼值 (Slacking):** 过高被开除，适度恢复精神。

## 3. 首批 MVP 职业 (Initial MVP Professions)
1. **全栈开发工程师:** 专注于 Bug、代码优雅度、删库跑路梗。
2. **新媒体运营:** 专注于 10w+ 爆款、0 预算挑战、账号切错梗。
3. **职场萌新 (管培生):** 专注于贴发票、取外卖、整顿职场梗。

## 4. 技术栈 (Tech Stack)
- **前端:** React (TypeScript) + Vite + Vanilla CSS (极致响应式与复古 UI)。
- **后端:** Go (Gin) - 负责鉴权、存档、排行榜。
- **数据管线:** Excel/CSV -> Python (Pandas) -> JSON (配置驱动开发)。
- **部署:** Nginx + Linux + MySQL/SQLite。

## 6. 核心规范 (Conventions)
- **文档优先:** 所有的重大变更必须记录在 `.gemini/` 目录下。
- **配置驱动:** 游戏逻辑与文案严禁硬编码。
- **UI 风格:** 采用复古/办公软件风，降低美术门槛，增强黑色幽默感。
